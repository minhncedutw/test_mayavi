# Mayavi
> Test Mayavi 3D visualization.



## Installation

Windows & OS X & Linux:
1. Install anaconda/miniconda
2. Install environment:

```sh
conda env create -f environment.yml
```

## Run sample
```sh
python 02_surface_visualization.py
```
